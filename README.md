# Movie
A Sample Movies App with the best practices for working with android architecture and tools.

#### Included: 
- Clean Architecture with Mvvm 
- Retrofit 
- Kotlin Coroutines and Flow 
- Android Navigation Component 
- ROOM Db 
- Picasso 
- Koin
- Animations
